# 09 — CALCULATE Advanced Patterns

Даний блок містить продвинуті патерни використання **CALCULATE()** у Power BI/DAX:
- зміна контексту фільтра;
- заміна фільтрів;
- очистка контексту;
- перетворення рядкових фільтрів у числові умови;
- застосування декількох фільтрів одночасно;
- умовні обчислення з контекстом;
- scenarios-based CALCULATE.

Формат:
- `examples.md` — 10+ готових прикладів;
- `patterns.md` — узагальнені патерни використання CALCULATE() з короткими поясненнями.

Цей блок призначений для виділення mastery-рівня роботи з контекстом у Power BI.
